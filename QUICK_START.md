# üöÄ Quick Start Guide - GitHub Desktop

## ‚úÖ Step-by-Step Instructions

### 1. Sign In to GitHub Desktop
- Open GitHub Desktop
- Sign in with your GitHub account
- If you don't have an account, create one at github.com

### 2. Add Your Project
1. Click **"File"** ‚Üí **"Add Local Repository"**
2. Click **"Create a New Repository"** button
3. Fill in:
   - **Name**: `new-year-wish-2026`
   - **Description**: "A beautiful New Year 2026 wish webpage"
   - **Local path**: `D:\NEW YEAR PROJECT`
   - **Uncheck** "Initialize this repository with a README"
4. Click **"Create Repository"**

### 3. Commit Your Files
1. You'll see all your files in GitHub Desktop
2. At the bottom left, write a commit message:
   ```
   Initial commit: New Year 2026 wish webpage
   ```
3. Click **"Commit to main"** button

### 4. Publish to GitHub
1. Click **"Publish repository"** button (top right)
2. In the popup:
   - **Uncheck** "Keep this code private" ‚úÖ (IMPORTANT for free Pages)
   - Repository name: keep as `new-year-wish-2026`
3. Click **"Publish Repository"**
4. Wait for it to upload (you'll see progress)

### 5. Make It Live (GitHub Pages)
1. Go to https://github.com
2. Find your repository: `new-year-wish-2026`
3. Click **"Settings"** (top menu)
4. Click **"Pages"** (left sidebar)
5. Under **"Source"**:
   - Select: **"Deploy from a branch"**
   - Branch: **"main"**
   - Folder: **"/ (root)"**
6. Click **"Save"**
7. Wait 1-2 minutes

### 6. Your Site is Live! üéâ
Your site will be at:
```
https://YOUR_USERNAME.github.io/new-year-wish-2026
```

Replace `YOUR_USERNAME` with your actual GitHub username.

## üìù Making Updates Later

1. Edit your files (HTML, CSS, JS)
2. Open GitHub Desktop
3. You'll see the changes
4. Write a commit message (e.g., "Added photos to memories")
5. Click **"Commit to main"**
6. Click **"Push origin"** (top right)
7. Your site updates automatically in 1-2 minutes!

## üé® Adding Photos

1. Create an `images` folder in your project
2. Add your photos there
3. Edit `index.html` - replace photo placeholders with:
   ```html
   <img src="images/your-photo.jpg" alt="Beautiful Moment" style="width: 100%; height: 200px; object-fit: cover; border-radius: 10px;">
   ```
4. Commit and push using GitHub Desktop

## ‚ùì Troubleshooting

**Can't find "Publish repository"?**
- Make sure you've committed your files first

**Pages not working?**
- Make sure repository is **Public** (not private)
- Wait 2-3 minutes after enabling Pages
- Check Settings ‚Üí Pages shows "Your site is live at..."

**Need help?**
- Check the full guide in `GITHUB_SETUP.md`
